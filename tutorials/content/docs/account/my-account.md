See your own usage, API keys, and password from one place.

<p class="rb-panel-actions"><a class="rb-panel-button" href="#" data-panel-route="/myaccount">Open My Account</a></p>

## Steps

1. Open "My account" from the sidebar.
2. Top cards show your used/remaining/total data and user limits.
3. Use the Daily usage chart with a date filter to review per-day traffic.
4. If allowed, Per node usage shows consumption per node.
5. In API Keys, create a key with 1/3/6/12 months or forever lifetime; delete old keys when you rotate.
6. In Change password, enter current + new password or use the random generator button.

{{< callout type="info" >}}
**Good to know**

- API keys are shown only once—copy immediately.
- Prefer short-lived keys; delete and recreate if you suspect sharing.
- If you lack self_api_keys or self_change_password, those cards stay hidden.
- Narrow the date range for charts if you want faster loading.
{{< /callout >}}
