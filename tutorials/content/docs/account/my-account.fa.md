برای دیدن مصرف ادمین، کلیدهای API و تغییر رمز از این صفحه استفاده کن.

<p class="rb-panel-actions"><a class="rb-panel-button" href="#" data-panel-route="/myaccount">باز کردن حساب من</a></p>

## مراحل

1. از منوی کنار روی «حساب من» بزن.
2. در کارت‌های بالا، مصرف و سقف حجم/تعداد یوزر خودت را می‌بینی (Used/Remaining/Total).
3. نمودار Daily usage را با فیلتر تاریخ بررسی کن تا مصرف روزانه‌ات را ببینی.
4. اگر دسترسی داری، در Per node usage مصرف روی هر نود را مشاهده کن.
5. در کارت API Keys می‌توانی کلید تازه با عمر ۱،۳،۶،۱۲ ماه یا دائمی بسازی و کلیدهای قدیمی را حذف کنی.
6. در کارت Change password، رمز فعلی و جدید را بزن یا از دکمه تولید رمز تصادفی استفاده کن.

{{< callout type="info" >}}
**نکته‌ها**

- کلید API فقط یک‌بار نمایش داده می‌شود؛ حتماً همان لحظه کپی کن.
- برای امنیت، عمر کلید را کوتاه بگذار و در صورت شک، کلید را حذف و دوباره بساز.
- اگر self_api_keys یا self_change_password نداشته باشی، آن کارت‌ها را نمی‌بینی.
- قبل از گزارش مصرف، بازه زمانی را کوچک کن تا نمودار سریع‌تر لود شود.
{{< /callout >}}
