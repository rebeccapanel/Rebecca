برای تمدید تاریخ یا افزایش حجم لازم نیست یوزر جدید بسازی.

## مراحل

1. در جدول کاربران روی نام کاربر یا آیکن ویرایش بزن تا پنجره تنظیمات باز بشه.
2. در بخش «تاریخ انقضا» روز جدید رو انتخاب کن یا عدد حجم رو زیاد کن.
3. اگر کاربر غیر فعال شده، وضعیت رو برگردون روی «active» یا «limited» بسته به سیاستت.
4. تغییرات رو ذخیره کن. لینک قبلی کاربر همچنان معتبر می‌مونه.

{{< callout type="info" >}}
**نکته‌ها**

- اگر می‌خوای بعد از تمام شدن حجم یا تاریخ، خودش تمدید بشه، بخش «Auto Renew» رو فعال کن.
- برای پاک کردن مصرف و شروع از صفر، دکمه reset usage توی پنجره کاربر هست.
{{< /callout >}}
