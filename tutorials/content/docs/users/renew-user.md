<span id="section-renew-user"></span>

You don’t need a brand new user to extend time or quota.

## Steps

1. In the table, click the username or edit icon to open the user drawer.
2. Pick a new expiry date or increase the data limit.
3. If the user was blocked, switch the status back to "active" (or "limited" if you want to cap them).
4. Save your changes. The existing links keep working.

{{< callout type="info" >}}
**Good to know**

- Turn on Auto Renew if you want the account to extend itself when it hits time or quota.
- Reset usage is available in the same dialog if you want to start counting from zero.
{{< /callout >}}
