When you need a one-off config for a user, grab it from the Links tab.

## Steps

1. On the Users page, click the edit icon for the user.
2. Open the "Links" tab.
3. Use the QR or copy button to grab the single config.

{{< callout type="info" >}}
**Good to know**

- QR is fastest for sharing to phones; copy is best for messaging.
- If the subscription link is broken, the single config works as a fallback.
{{< /callout >}}
