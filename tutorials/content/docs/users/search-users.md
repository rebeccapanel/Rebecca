<span id="section-user-search"></span>

You can paste almost anything from a user and find them instantly.

## Steps

1. Paste a subscription URL (token, key, or username/key) into the Users search box.
2. Paste a config link (vless/vmess/trojan/shadowsocks) to find the owner.
3. You can also search by UUID, credential key, or raw token.

{{< callout type="info" >}}
**Good to know**

- If a user cannot connect, paste their exact config link to locate the account quickly.
- Search accepts both plain values and full URLs.
{{< /callout >}}
