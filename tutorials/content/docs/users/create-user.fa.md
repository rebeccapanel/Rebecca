یوزر را سریع بساز و لینک را بده؛ نیاز به تنظیمات بک‌اند نیست.

<p class="rb-panel-actions"><a class="rb-panel-button" href="#" data-panel-route="/users">باز کردن کاربران</a><a class="rb-panel-button" data-primary="true" href="#" data-panel-route="/users" data-session-key="openCreateUser" data-session-value="true">ساخت یوزر</a></p>

## مراحل

1. در صفحه کاربران روی «ساخت یوزر» بزن تا دیالوگ باز شود.
2. نام کاربر، وضعیت (active یا on hold)، تاریخ انقضا و حجم را وارد کن.
3. سرویس و ورودی‌ها را انتخاب کن تا لینک درست ساخته شود.
4. محدودیت IP و یادداشت را تنظیم کن.
5. در صورت نیاز Auto Renew را فعال کن.
6. ذخیره کن و لینک یا QR را برای کاربر بفرست.

{{< callout type="info" >}}
**نکته‌ها**

- اگر می‌خواهی تحویل را آماده نگه داری، وضعیت را on hold بگذار تا بعد از اولین اتصال فعال شود.
- Note کوتاه بنویس تا بعداً یادت بماند این یوزر برای چه کاری است.
- محدودیت IP روی 1 یا 2 اشتراک‌گذاری را سخت‌تر می‌کند.
{{< /callout >}}
