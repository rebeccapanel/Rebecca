Use this when you want to hand over the link now but let activation wait for first connect.

## Steps

1. Set the status to "on hold" in the create/edit form.
2. Fill the "on hold duration" with 2-5 days so it expires if they never connect.
3. For safety, set the IP limit to 1 or 2.
4. Share the subscription link; after the first connection it becomes active and starts counting.

{{< callout type="info" >}}
**Good to know**

- If the on-hold window ends and they still haven’t connected, put it back on hold or delete it.
- Once it turns active, you can extend traffic or time from the same dialog.
{{< /callout >}}
