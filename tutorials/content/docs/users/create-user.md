Create a user fast and hand over the link—no backend work needed.

<p class="rb-panel-actions"><a class="rb-panel-button" href="#" data-panel-route="/users">Open users</a><a class="rb-panel-button" data-primary="true" href="#" data-panel-route="/users" data-session-key="openCreateUser" data-session-value="true">Create user</a></p>

## Steps

1. On the Users page, click “Create user” to open the dialog.
2. Fill username, status (active or on hold), expiry date, and data cap.
3. Choose service and inbounds so the right link is generated.
4. Set IP limit and add a note if needed.
5. Enable Auto Renew if you want hands-off renewals.
6. Save and share the link or QR with the user.

{{< callout type="info" >}}
**Good to know**

- Use on hold to keep it ready without starting the clock until first connect.
- Add a short note so you remember why this account exists.
- IP limit 1–2 makes sharing much harder.
{{< /callout >}}
