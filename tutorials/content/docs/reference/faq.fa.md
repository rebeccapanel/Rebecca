{{< details title="کی از on hold استفاده کنم؟" closed="true" >}}

<span id="faq-when-onhold"></span>

وقتی نمی‌خواهی یوزر بلافاصله فعال شود یا مطمئن نیستی مشتری کی می‌خواهد وصل شود.

- حساب‌های تستی که شاید هیچ‌وقت وصل نشوند.
- مشتریانی که اول باید هزینه را واریز کنند و بعد دسترسی بگیرند.
- وقتی می‌خواهی لینک را زود بفرستی اما شمارش ترافیک بعد از اتصال شروع شود.

{{< /details >}}

{{< details title="چطور سریع تمدید کنم؟" closed="true" >}}

<span id="faq-fast-renew"></span>

روی نام کاربر بزن، تاریخ یا حجم را عوض کن و ذخیره کن.

- برای تمدید خودکار، در همان پنجره گزینه Auto Renew را فعال کن.
- اگر مصرف صفر شد و می‌خواهی دوباره از اول بشمارد، reset usage بزن.

{{< /details >}}

{{< details title="Active، on hold و disabled چه فرقی دارند؟" closed="true" >}}

<span id="faq-status-differences"></span>

active یعنی دسترسی کامل؛ on hold یعنی در انتظار اولین اتصال؛ disabled یعنی یوزر را موقتا قطع کرده‌ای و خودش فعال نمی‌شود تا دوباره تغییر دهی.

{{< /details >}}

{{< details title="کاربر می‌گوید وصل نمی‌شود، از کجا شروع کنم؟" closed="true" >}}

<span id="faq-troubleshoot"></span>

سه مورد اول را چک کن: وضعیت کاربر (active باشد)، تاریخ و حجم (محدود نشده باشد)، و اینکه لینک قدیمی نباشد.

- از جدول کاربران یک QR یا لینک تازه بگیر.
- اگر وضعیت limited یا expired است، به اندازه لازم تمدید کن.
- در صورت شک به اشتراک‌گذاری، IP limit را کم کن و reset usage بزن.

{{< /details >}}

{{< details title="اشتراک چیست و کجا بسازم؟" closed="true" >}}

<span id="faq-subscription-basics"></span>

اشتراک همان یوزر است با لینک اتصال. از صفحه «کاربران» دکمه ساخت یوزر را بزن تا یک اشتراک بسازی.

- محدودیت حجم یعنی سقف ترافیک کاربر (GB).
- محدودیت زمان یعنی تاریخ انقضا؛ بعد از آن دسترسی قطع می‌شود.
- «سرویس» یعنی پلن آماده با تنظیمات از پیش‌تعریف‌شده (inbound، ترافیک و زمان).
- برای ساخت سریع، روی «ساخت یوزر» کلیک کن تا دیالوگ ساخت باز شود.

{{< /details >}}

{{< details title="سوالات رایج درباره تنظیمات اشتراک" closed="true" >}}

<span id="faq-subscription-advanced"></span>

اصطلاحات رایج هنگام ساخت یا ویرایش اشتراک:

- «حذف خودکار»: بعد از اتمام زمان/حجم کاربر حذف شود.
- «تمدید خودکار»: Auto Renew حجم/زمان را خودش شارژ می‌کند.
- «داده‌های اضافی»: همان add remaining است؛ ترافیک باقی‌مانده به دوره بعد می‌رود.
- «لغو دسترسی»: revoke subscription لینک‌های قبلی را بی‌اعتبار می‌کند.
- «بازنشانی مصرف»: reset usage شمارش مصرف را صفر می‌کند.
- «فعال/غیرفعال»: دسترسی را موقت وصل یا قطع می‌کند.
- «اشتراک چگونه متصل می‌شود؟»: از لینک/QR و کلاینت V2Ray/Reality/ShadowSocks.

{{< /details >}}
