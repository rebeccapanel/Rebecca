بخش مدیریت برای کارهای سراسری سیستم است. این بخش فقط به ادمین‌های `sudo` و `full_access` نمایش داده می‌شود، چون عملیات آن می‌تواند زیرساخت، مجوزها یا سرویس‌های مشترک را تغییر دهد.

## راهنماها

- [انتخاب خودکار سرویس با Service Inbound](service-auto-inbound/) — انتخاب سرویس بر اساس تگ اینباندها.
- [فعال‌سازی بکاپ تلگرام](telegram-backup/) — اتصال بات و زمان‌بندی بکاپ پنل.
- [نودها](nodes/) — ابتدا نود را در پنل بسازید، بسته نصب اختصاصی آن را بردارید و بعد سرور نود را متصل کنید.
- [ادمین‌ها](admins/) — ساخت ادمین و اعمال محدودیت‌ها.
- [نقش‌های ادمین](roles/) — شناخت سطح دسترسی هر نقش.

{{< callout type="warning" >}}
پیش از تغییر نودها، نقش‌ها یا تنظیمات مشترک، مقصد عملیات را دوباره بررسی کنید و یک بکاپ به‌روز داشته باشید.
{{< /callout >}}
