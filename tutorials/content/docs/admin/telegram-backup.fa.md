برای بکاپ‌گیری خودکار، اول ربات تلگرام را بساز و در پنل فعال کن، بعد گزینه Backup را روشن کن.

## مراحل

1. با BotFather یک ربات تلگرام بساز و توکنش را محرمانه نگه دار.
2. یک گروه خصوصی برای بکاپ‌ها بساز و ربات را داخل گروه اضافه کن.
3. در پنل اول ربات را فعال کن، بعد سراغ گزینه Backup برو.
4. یک بکاپ دستی تستی بگیر تا مطمئن شوی فایل‌ها داخل گروه ارسال می‌شوند.

{{< callout type="info" >}}
**نکته‌ها**

- گروه بکاپ را خصوصی نگه دار و فقط ادمین‌های مطمئن را اضافه کن.
- توکن ربات مثل رمز عبور است؛ آن را برای کاربران یا گروه عمومی نفرست.
- اگر پنل دکمه تست تلگرام دارد، قبل از فعال کردن بکاپ زمان‌بندی‌شده حتماً تست بگیر.
{{< /callout >}}

## ساخت ربات با BotFather {#section-telegram-backup-botfather}

BotFather ابزار رسمی تلگرام برای ساخت و مدیریت ربات‌هاست.

1. در تلگرام @BotFather را جستجو کن و وارد چت شو.
2. روی Start بزن یا دستور /newbot را بفرست.
3. یک نام نمایشی برای ربات وارد کن؛ مثلاً Rebecca Backup.
4. یک username یکتا وارد کن که آخرش bot باشد؛ مثلاً rebecca_backup_bot.
5. توکنی که BotFather می‌دهد را کپی کن و در جای امن نگه دار.

{{< callout type="info" >}}
**نکته‌ها**

- هرکس توکن را داشته باشد می‌تواند ربات را کنترل کند، پس آن را عمومی نکن.
{{< /callout >}}

## ساخت گروه بکاپ {#section-telegram-backup-group}

1. در تلگرام یک گروه جدید بساز و اسم واضحی مثل Panel Backups بگذار.
2. از تنظیمات گروه، گزینه Forum را فعال کن؛ گروه بکاپ باید حالت فروم داشته باشد.
3. رباتی که ساختی را به گروه اضافه کن.
4. اگر پنل برای ارسال فایل یا پیام نیاز دارد، ربات را ادمین گروه کن.
5. قبل از تست اتصال پنل، یک پیام معمولی داخل گروه بفرست.

{{< callout type="info" >}}
**نکته‌ها**

- گروه را خصوصی نگه دار تا فایل‌های بکاپ در دسترس افراد اضافه قرار نگیرد.
- اگر پنل Chat ID یا Group ID خواست، از بخش تست/تشخیص تلگرام داخل پنل استفاده کن؛ اگر چنین گزینه‌ای نبود، طبق راهنمای همان بخش مقدار را وارد کن.
{{< /callout >}}

## فعال کردن Backup در پنل {#section-telegram-backup-panel}

1. در پنل وارد تنظیمات Telegram/Bot شو و ربات را با توکن BotFather فعال کن.
2. با پیام تست مطمئن شو ربات می‌تواند به گروه بکاپ پیام بفرستد.
3. بعد مسیر Master settings -> Telegram -> <a href="#" data-panel-route="/integrations?focus=periodic-backup#telegram">Periodic Backup</a> را باز کن و بکاپ تلگرام را فعال کن.
4. تنظیمات را ذخیره کن و یک بکاپ دستی بگیر تا ارسال فایل داخل گروه را ببینی.
5. بعد از موفق بودن تست، بکاپ زمان‌بندی‌شده را روشن بگذار.

{{< callout type="info" >}}
**نکته‌ها**

- ترتیب مهم است: اول ربات را فعال کن، بعد گزینه Backup را روشن کن.
- اگر چیزی ارسال نشد، دسترسی ادمین ربات، Group ID و درست کپی شدن توکن را بررسی کن.
{{< /callout >}}
